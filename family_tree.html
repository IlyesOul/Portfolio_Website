Family Tree

Reading in data from a 2d Array Matrix, the program counstructs a "Family Tree" LinkedList with keeping generations intact. The user can then get a list of any persons relatives. This project uses doubly-linked lists and recursion. While a Binary/RB tree seems more intuitive, the program required two seperate pointers and didn't need to be sorted.



This program uses LinkedList, sets, recursion, and 2D-Arrays. While a binary tree seems intuitive, the program required two unique kinds of pointers and didn't need to be sorted



The average time complexity in O(n)
