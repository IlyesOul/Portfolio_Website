This page contains my most significant and advanced programs in Java

Family Tree of Unisprouts

This is family tree! This program essentially takes in a 2D Array of names and 'children', and contructs a LinkedList that retains the appropriate relationships between people.



Bucket-sort with Linked Lists

This is LL Bucketsort! The program will essentially take in a list of numbers (user-provided), and sort them using the Bucketsort algorithim. The catch is, however, that linked lists are used in combination with one array as opposed to a 2D Array!



Custom Tic-Tac-Toe

This is the usual Tic-Tac-Toe, but with a twist: only 'n' amount of x/o objects are allowed on the board at once, the user gets to decide how many pieces need to be in a row to win, and the user determines the dimensions of the board
