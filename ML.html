Prioritized To-Do List

This is my Prioritized To-Do list program! It essentially sorts a users to-do list by using training data (that the user inputs) and reminds the user every 'n' (user-provided) minutes to do an event (descending in significance)



This program uses sklearn, Recursive Feature Elimination, Random Forest Ensemble Classifiers, objects, and the plyer notificaiton library
